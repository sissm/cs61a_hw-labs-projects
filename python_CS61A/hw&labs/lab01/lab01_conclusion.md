# lab01_conclusion

## 学习总结

这次 `lab01` 主要练习了 Python 的基础函数、条件判断、循环和递归。我对这些内容有了更清楚的理解，尤其是下面几点：

- `return` 是把结果交给程序，`print` 是把内容显示出来。
- 数字题通常可以通过 `% 10` 和 `// 10` 来逐位处理。
- 循环题要先明确范围，再决定如何累加或判断。
- 递归题要特别注意终止条件，以及每一步如何逼近答案。

## 这几道题的收获

### 1. `digit`

`digit` 让我学会了如何取出整数中某一位的数字。通过整数除法和取模，可以方便地找到从右边数第 `k` 位。

### 2. `middle`

`middle` 让我练习了条件判断。对三个不同的数来说，找中间值的关键是写出正确的比较关系。

### 3. `falling`

`falling` 让我理解了递归的基本结构：如果问题变小后和原问题形式相同，就可以调用自己；如果已经到达最简单的情况，就直接返回结果。

### 4. `divisible_by_k`

`divisible_by_k` 让我更清楚地区分了“打印”和“返回”。函数不仅要输出满足条件的数字，还要返回它们的个数，这对我理解程序结构很有帮助。

### 5. `sum_digits`

`sum_digits` 让我熟悉了逐位拆分数字的方法。每次取个位，再去掉个位，就可以把所有数字累加起来。

### 6. `double_eights`

`double_eights` 让我学会了判断相邻数字是否相同。这类题的关键是比较当前位置和前一个位置，而不是只统计数字出现了几次。

## 最终结论

`lab01` 帮助我把 Python 的基础语法真正串起来了。我不仅理解了函数怎么写，也开始明白如何把题目拆成“输入、处理、输出”三个部分。以后遇到类似题目，我会先想清楚题目要什么，再选择合适的循环、条件或递归方法来实现。
